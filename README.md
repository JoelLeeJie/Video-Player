# Video Player

Showcase: https://youtu.be/Bcz6F4j5YMg

A full Audio and Visual Video Player that has basic features
- Choose different videos
- Seek forward/backward
- Pause/Play

Created using FFmpeg (video decoder) and SDL2 (output), in C++.
